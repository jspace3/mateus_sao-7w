## Instalação no Servidor

### Opção 1: Via Gerenciador de Arquivos (cPanel/Plesk)
1. Faça o upload da pasta `site-wordpress` para o seu servidor
2. Coloque os arquivos dentro da pasta public_html (ou www)
3. Acesse seu domínio - o site estará funcionando!

### Opção 2: Via FTP
1. Conecte-se ao seu servidor via FTP (FileZilla, etc)
2. Envie todos os arquivos para a pasta raiz do site
3. Mantenha a estrutura de pastas intacta
4. Pronto!
